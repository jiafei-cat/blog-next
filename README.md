## 技术栈

前端
- `create-next-app`创建项目
- `mobx/react-mobx-lite`做数据管理
- `date-fns`时间转换
- `antD`UI组件库
- `react-md-editor`MD富文本编辑
- `axios`请求库

后台
- `typeorm`数据库交互
- `mysql2`数据库连接
- `iron-session`管理生成session

## 项目运行
